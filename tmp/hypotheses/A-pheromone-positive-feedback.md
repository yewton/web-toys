# 仮説A: フェロモン正のフィードバック過剰

## 症状との対応

300kステップで「巣の形状がほぼ凍結」「既存トンネル内をアリが循環するだけ」になる主要因と推定。

## 該当箇所

- `ants-nest-simulator/src/Ant.ts:339` 帰巣中の追従:
  ```ts
  this.angle -= phDiff * 0.3;
  ```
- `ants-nest-simulator/src/Ant.ts:342` 探索中の回避:
  ```ts
  this.angle += phDiff * 0.1;
  ```
- `ants-nest-simulator/src/constants.ts`:
  - `PHEROMONE_DECAY = 0.999`（半減期 ≈ 693step）
  - `PHEROMONE_DEPOSIT_RETURN = 0.005`（探索の 5 倍）
- `ants-nest-simulator/src/grid.ts:164` 上限:
  ```ts
  state.pheromone[z][idx] = Math.min(1.0, ...)
  ```

## メカニズム

帰巣係数 0.3 vs 探索係数 0.1 の非対称 + 帰巣 deposit が 5 倍 + 緩い減衰 → 既存ルートのフェロモン濃度が指数的に強化される。上限 1.0 で飽和し、近傍が広く高フェロモン化 → アリが集中 → さらに強化、というポジティブフィードバック。

## 検証方法

### 計測1: フェロモン濃度ヒストグラム

シミュレーション中（10k/50k/100k/200k/300k step時点）に `state.pheromone[z]` 全セルを取得し、ヒストグラム（0.0–0.1, 0.1–0.3, 0.3–0.6, 0.6–1.0 の cell 数）をログ出力。

**仮説が正なら:** 後半ほど「上限近く（>0.6）」のセル数が増え、低濃度セルが減る。理想的には中濃度が広く分布するはず。

### 計測2: アリ位置のクラスタリング指標

各ステップで全アリの `(x, y)` の標準偏差・分散を出力。

**仮説が正なら:** ステップ数が進むにつれ分散が単調減少 or 振動の少ない狭い範囲に収束する。

### 介入実験

`PHEROMONE_DECAY = 0.996`, `PHEROMONE_DEPOSIT_RETURN = 0.003`, 上限を `0.3` に変えて再実行。同じチェックポイントでスクリーンショット比較。

**仮説が正なら:** 60k→300k の差分が「現状ほぼ変化なし」から「巣形状が継続的に変化」へ改善する。
