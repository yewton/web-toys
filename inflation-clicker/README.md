# インフレクリッカー (Inflation Clicker)

桁を上げて、天文学的な敵 HP を削り切るクリッカー。
削れていくキングダムハーツ風の分割体力ゲージが中心。

## 遊び方

1. コースを選んでスタート
2. 敵を連打して攻撃する
3. たまに出現するアイテムを取ると攻撃力（の桁）が伸びる
4. 累積ダメージの桁が敵 HP の桁に追いつけば撃破

攻撃力の伸びは漢字命数で表示される（万・億・…・無量大数・矜羯羅・阿伽羅…）。
進行状況はコースごとに `localStorage` へ自動保存。メニューの **つづきから** で再開できる。

## コース

到達する HP 指数の命数で命名。コース間の差異は HP のみ。

| コース | 総HP | 目安 |
|---|---|---|
| 無量大数 | 10⁶⁸ | 約 1 分 |
| 摩婆羅 | 10⁸⁹⁶ | 約 5 分 |
| 界分 | 10⁷¹⁶⁸ | 約 8 分 |
| 不可説不可説転 | 10^(7×2¹²²) ≈ 10³·⁷ˣ¹⁰³⁷ | 極限 2 時間 |
| グラハム数 ⚠️ | 10^(1.7×10³⁰⁸) | 連打で 1 週間級 |

⚠️ はメニューで警告枠として表示。事実上クリア不能ではないが、文字通り張り付くプレイになる。

> 補足: `不可説不可説転` の値は八十華厳（実叉難陀訳・大正蔵279）の定義 `10^(7×2¹²²)` を採用（四十華厳 `10^(7×2¹⁴²)`、六十「不可説転転」 `10^(10×2¹²⁰)` と異なる）。
